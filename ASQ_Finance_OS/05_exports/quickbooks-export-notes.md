# QuickBooks Export Notes

This file explains how records might later map into QuickBooks or another accounting system.

## Likely export fields

For expenses:

- date
- vendor
- amount
- currency
- category
- payment method
- memo/business purpose
- receipt location
- tax category guess

For revenue:

- date
- client/customer
- amount
- category
- invoice/payment reference
- notes

## Important

This repo is not a substitute for bank reconciliation.
A bookkeeper or accounting tool should eventually reconcile these records against actual accounts.
