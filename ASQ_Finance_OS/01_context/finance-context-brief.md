# Finance Context Brief

ASQ Ashlee is an AI systems, consulting, content, and business-building project.

This financial context system exists because Ashlee is accruing subscription costs while building ASQ Ashlee and related systems. Many costs are SaaS tools, AI tools, website tools, design tools, automation tools, domains, education, business operations, and client-delivery tools.

Ashlee has ADHD and should not be expected to maintain traditional spreadsheets manually. The system should prioritize:

- fast capture
- low-friction notes
- clean downstream structure
- visible upcoming charges
- emotionally neutral money clarity
- useful summaries for decision-making
- useful outputs for CPA/bookkeeper/tax/investor needs

## Business use cases

This system should support:

1. Knowing monthly ASQ software burn.
2. Avoiding accidental trial conversions.
3. Understanding annualized subscription costs.
4. Tracking deductible business expenses for CPA review.
5. Tracking revenue.
6. Supporting ASQ service pricing.
7. Producing draft monthly P&L summaries.
8. Preparing for QuickBooks, a bookkeeper, a CPA, grants, loans, or investors.

## Important framing

This is not shame accounting.
This is not a perfection spreadsheet.
This is financial context management.

The system should help Ashlee face financial reality in small, manageable pieces.
