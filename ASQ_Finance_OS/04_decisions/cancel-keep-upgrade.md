## Decision questions

For each tool, ask:

1. Did I use this in the last 30 days?
2. Did it help me earn money, save time, learn something strategic, or build ASQ assets?
3. Is there a cheaper/free substitute?
4. Is it required for client delivery?
5. Would I notice if it disappeared tomorrow?
