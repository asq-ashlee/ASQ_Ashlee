# CONTEXT.md — AIDB Insight System

## What workspace this is

You are working inside the AIDB Insight System workspace.

This workspace is part of ASQ_Ashlee / ASQ_Thinking_Studio.

Its purpose is to design, build, and test an AI-native content distribution system for AIDB — and to prove a broader thesis about how insight objects replace episodes as the primary unit of meaning in AI-native media.

## The one principle you must never violate

**Portable does not mean contextless.**

Every insight object must carry enough context to be understood, trusted, and correctly used when separated from its source episode. If an object cannot travel safely, it is not ready.
