FILE 09 — PRINTFUL PRICING
Last updated: 2026-05-21
Source: Printful API (base costs) + Printful shipping rate page (shipping costs)

## Margin Formula
Retail = (Printful base + shipping) / 0.40, rounded to nearest $5
Target margin: 60%
Free shipping to customer via Printful #PF-FRGFREE profile
Kaleigh absorbs shipping cost — shipping MUST be included in the retail price calculation

## Product Selection Rule
If a framed canvas size exists for an artwork → sell framed canvas only
If framed canvas is NOT available (panoramic artworks) → sell unframed canvas
Never offer both framed and unframed for the same size
