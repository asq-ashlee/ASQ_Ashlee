# Products Dataset Schema
# Arts of August
Last updated: 2026-05-21

## Purpose
Portable record of Arts of August product metadata.
One row per product variant.

## Primary File
product_records/AOA_PRD_[XXXX].csv

## Fields

### Identity
- product_id: stable Arts of August product ID. Format: AOA-PRD-[artwork#]-[variant#]
- artwork_id: reference to parent artwork. Format: AOA-ART-[XXXX]
- Product Name: full product name including size and frame color where applicable

### Product Classification
- Product Type: Original Painting / Printful Print
- Format: Original / Framed Canvas / Canvas Unframed
- Printful Product: select — Framed Canvas (in) / Canvas (in) / Thin Canvas (in)

### Dimensions
- Size: Printful catalog size. Format: 8×10 (no orientation — stored separately)
- Orientation: Horizontal / Vertical / Square
- Frame Color: White / Brown / Black / Framed (show frame) / N/A

### Pricing
- Printful Base Cost: number — Printful's base price before shipping
- Shipping Cost: number — Printful shipping rate for this size tier (US)
- Total Cost: number — Printful Base Cost + Shipping Cost
- Margin %: number — target margin percentage (default 60)
- Price: number — retail price shown to customer

### Fulfillment
- Fulfillment: Printful / Artist direct
- Shipping Model: Printful-free / Fixed-seller-ships
- Etsy Shipping Profile: text — profile name/code assigned in Etsy
- Lovable Shipping Rate: number — fixed $ shown at checkout (0 for Printful, fixed $ for originals)

### Sales Channels
- Sales Channels: multi-select — Etsy / Lovable / Market / Gallery
- Etsy Listing ID: text — populated when listing goes live
- Etsy Status: select — draft / active / inactive
- Lovable Status: select — draft / active / inactive

### Workflow
- Status: select — draft / ready-for-review / active / retired
- Notes: text — human context only. Not for operational data.

## Record Rules
- Each variant is one record (size × frame color = one record)
- product_id must be stable and never reused
- Printful SKU is generated by Printful and stored separately — never edit it
- Do not put operational data in Notes — use dedicated fields
- Lovable channel is for originals only
- Etsy channel is for Printful prints only
- No product is listed on both Lovable and Etsy simultaneously
- International shipping is not offered
